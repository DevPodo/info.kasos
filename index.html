Introduction

Welcome to the KasOS documentation. KasOS is a modern web-based desktop environment that provides a complete workspace solution with built-in cryptocurrency integration, security features, and extensible application support.

What is KasOS?

KasOS is an open-source desktop environment that runs in your browser. It provides a complete workspace with:

  • Full window management system
  • Built-in cryptocurrency wallet integration
  • Secure messaging capabilities
  • Development tools
  • Extensible application framework

Complete windowing system with taskbar, start menu, and multiple desktop support.

üõ°Ô∏è Security First

Desktop Environment

Features

  • Multiple virtual desktops: KasOS supports multiple desktops, allowing you to organize open windows and applications into separate workspaces. This feature is available in the main desktop interface and can be accessed via the desktop switcher.
  • Start menu with application categories: The Start menu provides access to all built-in and installed apps, organized by category. You can search and launch apps from here.
  • Taskbar with running applications: The taskbar at the bottom of the screen shows all currently running apps and windows. You can switch, minimize, or restore apps from the taskbar.
  • System tray with clock and status indicators: The system tray displays the current time and status icons for network and notifications. Some apps may add their own icons for background tasks.
  • Customizable wallpapers and themes: KasOS allows you to change your desktop wallpaper and switch between available color themes, including a cyberpunk theme.
  • Drag-and-drop support: You can move windows around the desktop by dragging their headers. File drag-and-drop is supported in the file manager and compatible apps.
  • Keyboard shortcuts: KasOS provides keyboard shortcuts for window management (move, snap, minimize, switch), app launching, and navigation.
  • Notifications: System and app notifications appear in a notification center accessible from the system tray.

Common Issues

Performance Issues

Symptoms: Slow application loading, UI lag, high memory usage

Solutions:
  • Close unused applications and windows
  • Clear browser cache and cookies
  • Disable browser extensions temporarily
  • Check available RAM and close other browser tabs
  • Use incognito/private browsing mode

Application Won't Load

Symptoms: App icon appears but window doesn't open

Solutions:
  • Check browser console for JavaScript errors
  • Refresh the page and try again
  • Clear browser storage and restart KasOS
  • Try loading the app in a different browser

Wallet Connection Issues

Symptoms: Can't connect to KasWare or other wallets

Solutions:
  • Ensure wallet extension is installed and enabled
  • Check that you're logged into your wallet
  • Refresh both KasOS and your wallet extension
  • Try connecting in incognito mode
  • Check wallet extension permissions

App Development

Getting Started

KasOS allows you to create custom applications using JavaScript. Each app runs in a sandboxed environment and is registered with the KasOS app registry. To get started, you should:

  1. Set up your development environment: Make sure you have Node.js and npm installed. Clone the KasOS repository and install dependencies as described in the Getting Started section.
  2. Create your app module: Write your app as a JavaScript module. Your app should export an object with at least an id, name, and init function.
  3. Register your app: Use window.KasOS.registerApp() to register your app with the system. This makes it available in the KasOS app menu.
  4. Test your app: Launch KasOS in development mode (npm run dev) and open your app from the desktop environment.
  5. Distribute your app: You can share your app module with others, or contribute it to the KasOS app registry.

Apps can interact with the KasOS API to open windows, access files, communicate with other apps, and more. For advanced features, see the API Reference section. 

// Example app registration
window.KasOS.registerApp({
  id: 'my-app',
  name: 'My App',
  init: function() {
    // App logic here
  }
});

App Architecture Patterns

KasOS apps follow modular architecture patterns for maintainability and performance:

MVC Pattern

// Model - Data management
class AppModel {
    constructor() {
        this.data = {};
        this.observers = [];
    }
    
    update(key, value) {
        this.data[key] = value;
        this.notifyObservers();
    }
    
    notifyObservers() {
        this.observers.forEach(observer => observer.update(this.data));
    }
}

// View - UI management
class AppView {
    constructor(container) {
        this.container = container;
    }
    
    render(data) {
        this.container.innerHTML = this.template(data);
        this.bindEvents();
    }
    
    template(data) {
        return `<div class="app-content">${data.content}</div>`;
    }
}

// Controller - Business logic
class AppController {
    constructor(model, view) {
        this.model = model;
        this.view = view;
        this.model.observers.push(this.view);
    }
}

    }

        // Override in subclasses
    }
}

State Management

// Global state management for complex apps
class StateManager {
    constructor() {
        this.state = {};
        this.subscribers = new Map();
    }
    
    subscribe(key, callback) {
        if (!this.subscribers.has(key)) {
            this.subscribers.set(key, []);
        }
        this.subscribers.get(key).push(callback);
    }
    
    setState(key, value) {
        this.state[key] = value;
        if (this.subscribers.has(key)) {
            this.subscribers.get(key).forEach(callback => callback(value));
        }
    }
    
    getState(key) {
        return this.state[key];
    }
}

// Usage in app
const appState = new StateManager();
appState.subscribe('user', (user) => {
    console.log('User updated:', user);
});
appState.setState('user', { name: 'John', email: 'john@example.com' });

// Virtual scrolling implementation
class VirtualScroll {
    constructor(container, itemHeight, items) {
        this.container = container;
        this.itemHeight = itemHeight;
        this.items = items;
        this.visibleItems = [];
        this.setupScrollListener();
    }
    
    setupScrollListener() {
        this.container.addEventListener('scroll', this.handleScroll.bind(this));
    }
    
    handleScroll() {
        const scrollTop = this.container.scrollTop;
        const visibleCount = Math.ceil(this.container.clientHeight / this.itemHeight);
        const startIndex = Math.floor(scrollTop / this.itemHeight);
        
        this.renderVisibleItems(startIndex, visibleCount);
    }
}

// Enhanced debugging utilities
class DebugUtils {
    static log(level, message, data = null) {
        const timestamp = new Date().toISOString();
        const output = `[${timestamp}] ${level.toUpperCase()}: ${message}`;
        
        console.log(output);
        if (data) console.log(data);
        
        // Send to KasOS debug panel
        window.KasOS?.debug?.log(level, message, data);
    }
    
    static profile(name, fn) {
        const start = performance.now();
        const result = fn();
        const end = performance.now();

        return result;
    }

        if (performance.memory) {
            return {
                used: Math.round(performance.memory.usedJSHeapSize / 1048576),
                total: Math.round(performance.memory.totalJSHeapSize / 1048576),
                limit: Math.round(performance.memory.jsHeapSizeLimit / 1048576)
            };
        }
        return null;
    }
}

Error Handling Best Practices

// Global error handling
class ErrorHandler {
    constructor() {
        this.setupGlobalHandlers();
    }
    
    setupGlobalHandlers() {
        window.addEventListener('error', this.handleError.bind(this));
        window.addEventListener('unhandledrejection', this.handlePromiseRejection.bind(this));
    }
    
    handleError(event) {
        const error = {
            message: event.message,
            filename: event.filename,
            lineno: event.lineno,
            colno: event.colno,
            stack: event.error?.stack
        };
        
        this.reportError('javascript', error);
    }
    
    handlePromiseRejection(event) {
        const error = {
            reason: event.reason,
            promise: event.promise
        };
        
        this.reportError('promise', error);
        event.preventDefault(); // Prevent console spam
    }
    
    reportError(type, error) {
        // Log to console
        console.error(`[${type.toUpperCase()}]`, error);
        
        // Send to KasOS error system
        window.KasOS?.errorSystem?.report(type, error);
        
        // Optional: Send to external logging service
        this.sendToLogger(type, error);
    }
}

// Initialize error handling
const errorHandler = new ErrorHandler();

// WASM Sandbox for secure code execution
class WASMSandbox {
    constructor() {
        this.modules = new Map();
        this.instances = new Map();
    }
    
    async loadModule(name, wasmBytes) {
        const module = await WebAssembly.compile(wasmBytes);
        this.modules.set(name, module);
        return module;
    }
    
    async createInstance(moduleName, imports = {}) {
        const module = this.modules.get(moduleName);
        if (!module) {
            throw new Error(`Module ${moduleName} not found`);
        }
        
        const instance = await WebAssembly.instantiate(module, {
            env: {
                memory: new WebAssembly.Memory({ initial: 1 }),
                log: (ptr, len) => {
                    const bytes = new Uint8Array(instance.exports.memory.buffer, ptr, len);
                    console.log(new TextDecoder().decode(bytes));
                },
                ...imports
            }
        });
        
        const instanceId = this.generateId();
        this.instances.set(instanceId, instance);
        return instanceId;
    }
    
    execute(instanceId, functionName, ...args) {
        const instance = this.instances.get(instanceId);
        if (!instance) {
            throw new Error(`Instance ${instanceId} not found`);
        }
        
        return instance.exports[functionName](...args);
    }
}

// Virtual DOM for efficient updates
class VirtualDOM {
    constructor() {
        this.tree = null;
    }
    
    createElement(type, props = {}, ...children) {
        return {
            type,
            props,
            children: children.flat()
        };
    }
    
    diff(oldTree, newTree) {
        const patches = [];
        this.diffNodes(oldTree, newTree, 0, patches);
        return patches;
    }
    
    patch(container, patches) {
        patches.forEach(patch => {
            switch (patch.type) {
                case 'CREATE':
                    container.appendChild(this.render(patch.node));
                    break;
                case 'REMOVE':
                    container.removeChild(container.childNodes[patch.index]);
                    break;
                case 'REPLACE':
                    container.replaceChild(
                        this.render(patch.node),
                        container.childNodes[patch.index]
                    );
                    break;
                case 'UPDATE':
                    this.updateElement(container.childNodes[patch.index], patch.props);
                    break;
            }
        });
    }
}

// Debouncing expensive operations
function debounce(func, wait) {
    let timeout;
    return function executedFunction(...args) {
        const later = () => {
            clearTimeout(timeout);
            func(...args);
        };
        clearTimeout(timeout);
        timeout = setTimeout(later, wait);
    };
}

// Throttling for scroll events
function throttle(func, limit) {
    let inThrottle;
    return function() {
        const args = arguments;
        const context = this;
        if (!inThrottle) {
            func.apply(context, args);
            inThrottle = true;
            setTimeout(() => inThrottle = false, limit);
        }
    }
}

// Offload heavy computations to Web Workers
class WorkerPool {
    constructor(workerScript, poolSize = 4) {
        this.workers = [];
        this.taskQueue = [];
        
        for (let i = 0; i < poolSize; i++) {
            this.workers.push({
                worker: new Worker(workerScript),
                busy: false
            });
        }
    }
    
    async executeTask(data) {
        return new Promise((resolve, reject) => {
            const availableWorker = this.workers.find(w => !w.busy);
            
            if (availableWorker) {
                availableWorker.busy = true;
                availableWorker.worker.postMessage(data);
                availableWorker.worker.onmessage = (e) => {
                    availableWorker.busy = false;
                    resolve(e.data);
                };
            } else {
                this.taskQueue.push({ data, resolve, reject });
            }
        });
    }

// Performance measurement utilities
class PerformanceMonitor {
    static measureFunction(fn, name) {
        const start = performance.now();
        const result = fn();
        const end = performance.now();
        console.log(`${name} took ${end - start} milliseconds`);
        return result;
    }
    
    static trackFrameRate() {
        let frames = 0;
        let lastTime = performance.now();
        
        function update() {
            frames++;
            const currentTime = performance.now();
            
            if (currentTime >= lastTime + 1000) {
                const fps = Math.round((frames * 1000) / (currentTime - lastTime));
                KasOS.system.updateMetric('fps', fps);
                
                // Log warning for low FPS
                if (fps < 30) {
                    console.warn(`Low FPS detected: ${fps}`);
                }
                
                frames = 0;
                lastTime = currentTime;
            }
            
            requestAnimationFrame(update);
        }
        
        update();
    }
    
    static trackLongTasks() {
        // Monitor for JavaScript long tasks (>50ms)
        const observer = new PerformanceObserver((list) => {
            for (const entry of list.getEntries()) {
                if (entry.duration > 50) {
                    console.warn(`Long task detected: ${entry.duration}ms`);
                    KasOS.system.reportLongTask(entry);
                }
            }
        });
        
        observer.observe({ entryTypes: ['longtask'] });
    }
}

// Efficient caching strategy
const CACHE_NAME = 'kasos-v1.2.0';
const STATIC_CACHE = 'kasos-static-v1';
const DYNAMIC_CACHE = 'kasos-dynamic-v1';

const urlsToCache = [
    '/',
    '/system.js',
    '/styles_clean.css',
    '/icons/kasos-icon-192.png',
    '/applications/core-apps.bundle.js'
];

self.addEventListener('install', event => {
    event.waitUntil(
        caches.open(STATIC_CACHE)
            .then(cache => cache.addAll(urlsToCache))
            .then(() => self.skipWaiting())
    );
});

self.addEventListener('fetch', event => {
    event.respondWith(
        caches.match(event.request)
            .then(response => {
                if (response) {
                    return response;
                }
                
                return fetch(event.request)
                    .then(fetchResponse => {
                        // Cache dynamic content
                        if (event.request.url.includes('/applications/')) {
                            const responseClone = fetchResponse.clone();
                            caches.open(DYNAMIC_CACHE)
                                .then(cache => cache.put(event.request, responseClone));
                        }
                        return fetchResponse;
                    });
            })
    );
});

// Local Storage Helper
class DataManager {
    constructor(appName) {
        this.prefix = `kasos_${appName}_`;
    }
    
    save(key, data) {
        try {
            localStorage.setItem(this.prefix + key, JSON.stringify(data));
            return true;
        } catch (error) {
            console.error('Save failed:', error);
            return false;
        }
    }
    
    load(key, defaultValue = null) {
        try {
            const data = localStorage.getItem(this.prefix + key);
            return data ? JSON.parse(data) : defaultValue;
        } catch (error) {
            console.error('Load failed:', error);
            return defaultValue;
        }
    }
    
    remove(key) {
        localStorage.removeItem(this.prefix + key);
    }
    
    clear() {
        Object.keys(localStorage)
            .filter(key => key.startsWith(this.prefix))
            .forEach(key => localStorage.removeItem(key));
    }
}

// Usage
const data = new DataManager('my-app');

class APIClient {
    constructor(baseURL) {
        this.baseURL = baseURL;
        this.headers = {
            'Content-Type': 'application/json'
        };
    }
    
    async get(endpoint) {
        try {
            const response = await fetch(`${this.baseURL}${endpoint}`, {
                method: 'GET',
                headers: this.headers
            });
            
            if (!response.ok) {
                throw new Error(`HTTP ${response.status}: ${response.statusText}`);
            }
            
            return await response.json();
        } catch (error) {
            console.error('GET request failed:', error);
            throw error;
        }
    }
    
    async post(endpoint, data) {
        try {
            const response = await fetch(`${this.baseURL}${endpoint}`, {
                method: 'POST',
                headers: this.headers,
                body: JSON.stringify(data)
            });
            
            if (!response.ok) {
                throw new Error(`HTTP ${response.status}: ${response.statusText}`);
            }
            
            return await response.json();
        } catch (error) {
            console.error('POST request failed:', error);
            throw error;
        }
    }
}

// Smooth Animations
class AnimationHelper {
    static fadeIn(element, duration = 300) {
        element.style.opacity = '0';
        element.style.display = 'block';
        
        return this.animate(element, { opacity: 1 }, duration);
    }
    
    static fadeOut(element, duration = 300) {
        return this.animate(element, { opacity: 0 }, duration)
            .then(() => {
                element.style.display = 'none';
            });
    }
    
    static slideIn(element, direction = 'left', duration = 300) {
        const transform = direction === 'left' ? 'translateX(-100%)' : 'translateY(-100%)';
        element.style.transform = transform;
        
        return this.animate(element, { transform: 'translate(0, 0)' }, duration);
    }
    
    static animate(element, properties, duration) {
        return new Promise((resolve) => {
            const startTime = performance.now();
            const startValues = {};
            
            // Get starting values
            Object.keys(properties).forEach(prop => {
                startValues[prop] = getComputedStyle(element)[prop];
            });
            
            function step(currentTime) {
                const elapsed = currentTime - startTime;
                const progress = Math.min(elapsed / duration, 1);
                
                // Easing function (ease-out)
                const eased = 1 - Math.pow(1 - progress, 3);
                
                // Apply interpolated values
                Object.keys(properties).forEach(prop => {
                    const start = parseFloat(startValues[prop]) || 0;
                    const end = parseFloat(properties[prop]);
                    const current = start + (end - start) * eased;
                    element.style[prop] = current + (prop.includes('opacity') ? '' : 'px');
                });
                
                if (progress < 1) {
                    requestAnimationFrame(step);
                } else {
                    resolve();
                }
            }
            
            requestAnimationFrame(step);
        });
    }
}
